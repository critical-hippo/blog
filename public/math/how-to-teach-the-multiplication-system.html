How to Teach The Multiplication System

  1. Introduction

    It's easy to teach how the long multiplication system works. But as easy as it is, none of my teachers from elementary school ever taught it to me. Honestly, it would be great if everyone knew how it did. As some Indian guy said, "Be the change you wish to see in the world."

  2. Digits After a Decimal Point

    Before we start, it's necessary to remember that decimal points function like ordinary digits. They're just after a dot. The most important part about them is remembering where the decimal point should be. That's why 1.22 is 1.44 and 122 is 144.

  3. How Carrying Works

    When adding or multiplying, we often get a number higher than 9. Of course, this gives us a two-digit number that we can not add to the digit in the ones' place. This is due to our number system's limitations.

    As you can see, you cannot add it to the ones' place. Conveniently, however, the number at the tens place of the answer can take that number because the excess digit is in the tens place.

  4. The Main Part: Multiplication

    I once forgot the system's direction while I was multiplying with a two-digit multiplier. I felt embarrassed as I needed to look it up. But when I discovered how it worked, I was never confused again.

  5. Finishing words

    We should teach these not as a set of rules but for what it is. If you ignore the words, it only took two images to show how multiplication works, which says a lot. Teachers should spend 5-20 minutes explaining how it works. That alone is enough.

    It may be enough for some, but it's not the best we can do. If you've ever heard of the Trachtenberg system for multiplication, you'll see how much simpler it is.
